# Simple Todo App with AngularJS

This is a simple to-do application built using AngularJS. The app allows you to add, edit, and delete tasks, as well as mark tasks as completed.

## Tech Stack
1. AngularJS 1.8.2
2. HTML
3. CSS
4. UI-Router
